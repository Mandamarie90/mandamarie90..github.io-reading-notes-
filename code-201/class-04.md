#### What is meant by “normal flow”?

It refers to the default positioning and layout behavior of HTML elements on a webpage.

#### What are a few differences between block-level and inline elements?

Block-level elements typically start on a new line and take up the full width available, while inline elements flow within the content and do not force a new line.

#### ___ positioning is the default for every html element.

Static positioning

#### Name a few advantages to using absolute positioning on an element.

Precise control over element placement, the ability to layer elements over one another, and independence from surrounding content flow.

#### What is a key difference between fixed positioning and absolute positioning?

Fixed positioning is relative to the viewport and remains fixed in its position even when the page is scrolled, while absolute positioning is relative to the nearest positioned ancestor and moves with scrolling if its ancestor is not fixed.

#### Describe the difference between a function declaration and a function invocation.


A function declaration is where you define a function, specifying its name, parameters, and code block, while a function invocation is when you actually call or execute the function by using its name followed by parentheses and passing any required arguments.

#### What is the difference between a parameter and an argument?


A parameter is a variable in a function definition, while an argument is the actual value passed to the function when it is called.
